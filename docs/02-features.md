# 02 · 功能特性清单(需求归档)

> 汇总三个参考实现的全部功能,按优先级分级。用作自研排期的需求池。
> 图例:✅ 三家都有(必备) · 🔶 部分有(进阶) · 🌟 国内特色 · 🧩 锦上添花

## A. MVP(最小可玩 —— 没有这些就不叫启动器)

| # | 功能 | 说明 | 级别 |
|---|------|------|------|
| A1 | 版本列表获取 | 拉 Mojang version manifest,展示可下载版本 | ✅ |
| A2 | 原版安装 | 下载 client jar + libraries + assets + version json | ✅ |
| A3 | 文件校验补全 | 大小 + SHA1 校验,缺失/损坏自动重下 | ✅ |
| A4 | natives 解压 | 平台相关本地库解压到临时目录 | ✅ |
| A5 | 离线登录 | 输入用户名直接玩(假 UUID) | ✅ |
| A6 | 微软正版登录 | OAuth → Xbox → MC profile 完整链路 | ✅ |
| A7 | Java 检测 | 扫描本机 Java,匹配版本 | ✅ |
| A8 | 命令行拼接 + 启动 | 占位符替换、classpath 拼接、起进程 | ✅ |
| A9 | 实例列表 | 列出已安装版本,选择启动 | ✅ |
| A10 | 内存/JVM 参数设置 | 至少能设 -Xmx 和自定义 JVM 参数 | ✅ |

## 推荐排期

```
Sprint 1 (能启动原版正版/离线)       → A 全部
Sprint 2 (能装 mod loader 玩模组)     → B1,B2,B5,B6,B7
Sprint 3 (国内可用 + 下载体验)        → C1,C2,C3,B8,B9
Sprint 4 (Mod/整合包生态)             → B10~B14,C4
Sprint 5 (体验完善)                   → B15~B21,C5,D1,D5
Sprint 6+ (差异化)                    → C6,D2~D8,E*
```

> 各功能的实现细节散落在 `modules/*` 各文档,需要做某项时去对应模块文档 + `refs/*` 找源码参考。
