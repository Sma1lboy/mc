FEATURES

你要的,它都会

从装游戏到玩上 Mod,中间的麻烦事全部替你做完。

一键开玩

挑好版本点 PLAY,游戏文件自动下载校验,下完直接进游戏 —— 全程不用你动手。

想玩什么 Mod 都行

Fabric、Quilt、Forge、NeoForge 全都支持,选一个自动装好,不用看教程折腾。

Java?不用管

缺什么版本的 Java 它自己发现、自己下载,你甚至不需要知道 Java 是什么。

怎么登录都可以

微软正版、离线模式、第三方外置登录,想怎么进就怎么进,多账号随时切换。

Mod 商店内置

直接在启动器里搜 Mod、一键安装,连它依赖的其他 Mod 也一起装好,再也不会缺前置。

存档与整合包

世界备份、重命名、回收站防误删;整合包一键导入导出,和朋友共享一套配置。

崩了?告诉你为什么

游戏崩溃时自动分析日志,用人话告诉你可能的原因和怎么修,不用对着报错发呆。

国内也快

内置国内下载加速,不挂工具也能满速下游戏和资源。

多开不占地方

建多少个游戏实例都行,共用的文件只存一份,硬盘不为重复文件买单。

Apple Silicon 与 Intel 双架构。推荐 Homebrew 安装,更新只要 brew upgrade。

Windows.MSI

标准 MSI 安装包,从 GitHub Releases 获取最新版本。

GET .MSI
Linux.APPIMAGE

AppImage 免安装直接运行,chmod +x 即用。

GET .APPIMAGE
