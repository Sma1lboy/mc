Windows 安装 / 测试提示

  1. 下载后双击安装。应用未签名,SmartScreen 可能拦截 → 点「更多信息 → 仍要运行」即可。
  2. 首次登录 kobeMC 账号(右上角)才会出现 好友 / 通知 / 领域 等社交功能。
  3. 玩联机大厅:领域页点「开启联机」时会弹管理员授权(需要建虚拟网卡)——这是正常的;可点「免密一键」只授权一次。
  4. 测两台机:房主进游戏「对局域网开放」后,成员领域页会出现「加入游戏」直接进服。同一台机两个实例测不了联机(需两台机 / 一个 VM)。
  5. 有问题把 设置 → 诊断 → 打开日志目录 里的 mc-launcher.log 发我。
